# Kamal.NET

A .NET port of [Kamal](https://github.com/basecamp/kamal) — deploy web apps anywhere, from bare metal to cloud VMs, with zero downtime. Same `config/deploy.yml`, same commands, no Ruby required.

```
dotnet tool install -g mvdmio.Kamal
kamal init
kamal setup
kamal deploy
```
